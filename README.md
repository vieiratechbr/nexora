# Nexora 🚀

Nexora é um site institucional desenvolvido para uma **startup fictícia de tecnologia**, criado exclusivamente para **fins de portfólio pessoal**.

O projeto tem como objetivo demonstrar habilidades em **HTML, CSS e JavaScript**, com foco em **design responsivo**, **boas práticas de UI/UX** e **interações dinâmicas no front-end**.

---

## 🧠 Sobre o Projeto

Nexora representa uma startup que oferece soluções digitais para pequenas e médias empresas, como:

* Desenvolvimento de websites
* Sistemas web simples
* Soluções digitais personalizadas

Todo o conteúdo, branding e identidade visual apresentados neste projeto são **100% fictícios** e **não possuem finalidade comercial**.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** — estrutura semântica e acessível
* **CSS3** — estilização, layout e design responsivo
* **JavaScript (Vanilla)** — interatividade e comportamento dinâmico
* **ScrollReveal.js** — animações baseadas em rolagem
* **Boxicons** — biblioteca de ícones utilizada na interface

---

## 📱 Funcionalidades

* Layout totalmente responsivo (abordagem mobile-first)
* Menu de navegação mobile interativo
* Animações suaves baseadas em scroll
* Estrutura moderna de site institucional
* Experiência de usuário limpa e intuitiva
